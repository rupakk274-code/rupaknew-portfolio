A full-stack portfolio project developed as part of academic coursework.
## Description
This is a personal portfolio website developed as a single-page application. It showcases my profile, skills, and project work.

The application includes a frontend built using React and a backend developed using Node.js and Express. The contact form is connected to a MySQL database to store user messages.

---

## Features
- Single page responsive design
- Navigation with smooth scrolling
- Projects and skills display
- Contact form integration
- Backend API for data handling

---

## Technologies Used
Frontend:
- React
- HTML, CSS, JavaScript

Backend:
- Node.js
- Express.js

Database:
- MySQL
